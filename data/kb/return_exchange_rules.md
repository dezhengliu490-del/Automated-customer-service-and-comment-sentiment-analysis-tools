# 退换货规则政策 (Return & Exchange Rules)

## 1. 总则与基本规定
- **七天无理由退换货：** 消费者在签收商品之日起7天内（含7天），在保证商品完好、不影响二次销售的前提下，可申请无理由退换货。
- **质量问题退换货：** 如商品存在非人为损坏的质量问题，消费者可自签收之日起15天内申请退换货，超出部分部分将依据国家“三包”政策执行。

## 2. 支持退换货的条件 (必须同时满足)
- 商品未经使用、洗涤、穿着、拆卸，无明显污渍或人为受损。
- 商品的外包装（如鞋盒、专用密封袋、防伪标签）、吊牌、说明书完整且未被撕毁或涂改。
- 附带的赠品、发票凭证等必须随主商品一并退回。

## 3. 不支持无理由退换货的商品类别
根据相关法律法规及行业规范，以下商品不支持七天无理由退换货：
1. **定制类商品：** 根据消费者个人需求进行定制、刻字、尺寸修改等特殊加工的商品。
2. **生鲜易腐商品：** 鲜活易腐烂的水果、海鲜、绿植等。
3. **数字化商品：** 在线下载或者消费者拆封的音像制品、计算机软件等数字化商品。
4. **交付的报纸、期刊等读物。**
5. **拆封后影响二次销售或人身安全的商品：** 如已拆封的贴身衣物（内衣裤、袜子）、化妆品、食品、保健品等。

## 4. 运费承担原则
- **非商家责任（买家个人原因）：** 尺码不合适、不喜欢、拍错等导致的七天无理由退换货，**退回运费及换货发出运费由买家自行承担**。若商品包邮，买家仅需承担退回运费；如是非包邮商品，买家需承担来回双程运费。
- **商家责任（质量问题/错发漏发）：** 因商品本身存在质量问题，或由于商家失误导致的错发、漏发，引发的退换货，**来回双程运费均由商家承担**。

## 5. 退换货操作流程
1. **发起申请：** 买家需在订单页面提交“退款/退货”申请，并选择相应的退换货原因。如因质量问题退货，请务必上传清晰的商品照片或视频作为凭证。
2. **商家审核：** 商家客服将在申请提交后的24-48小时内进行人工审核。
3. **退回商品：** 审核通过后，请买家在7天内将商品通过快递寄回，并在系统中填写正确的退货物流单号。（注：**请勿使用顺丰到付、平邮或其他不包邮特快专递拒收件返回**，否则可能导致仓库拒收）
4. **仓库签收与质检：** 商家仓库收到退货包裹后，将进行开箱质量检验（通常需要1-3个工作日）。
5. **处理退款/发货：**
   - **退款流程：** 质检确认无误后，系统将在1-3个工作日内将款项原路退回至买家付款账户。
   - **换货流程：** 质检通过后，仓库将按买家系统指定的新尺码/颜色重新发出包裹，并以短信或留言方式通知买家新的物流单号。

## 6. 特别注意事项
- 退回包裹内请务必放入手写便签，清晰写明您的**订单编号、联系电话、退换货原因（若为换货请求，请写明需要调换的对应颜色和尺码）**，以便仓库快速定位并为您优先排单处理。
- 请买家将贵重物品或易碎物品妥善打包加固，若在退回途中因包装不当导致破损而影响二次销售，相关损失需买家自行与物流公司协商理赔解决。
