本项目名为“自动化客服与评论情感分析系统”，专为中小电商卖家及本地生活商家设计，旨在解决海量客户咨询响应慢、海量用户评价难以高效分析的核心痛点。
本项目基于大语言模型（LLM）构建，集成了两项核心能力：
智能拟人化客服：引入检索增强生成（RAG）技术，系统能够外挂商家的“标准话术库”与“退换货规则”知识库，自动生成准确、合规且高度模仿该商家专属语气的客服回复，大幅降低人工客服的沟通成本。
深度评论情感分析：结合异步并发机制与大模型的文本理解能力，系统能够对海量原始评论进行高效的数据清洗、多维度情感分类（好/中/差评），并自动提取高频产品缺陷实体（如“包装破损”、“卡扣易断”等）。
最终，所有数据处理结果与商业洞察将通过直观的可视化交互面板（Dashboard）清晰呈现。本项目不仅交付了一套高可用性的工程原型系统，更能够为商家提供数据驱动的产品迭代建议，具有显著的学术研究与实际应用价值。

This project, named the "Automated Customer Service and Review Sentiment Analysis System," is specifically designed for small and medium-sized e-commerce sellers and local businesses. It aims to address the core pain points of slow responses to massive customer inquiries and the difficulty of efficiently analyzing voluminous user reviews.
Built upon Large Language Models (LLMs), this project integrates two core capabilities:
Intelligent Anthropomorphic Customer Service: By incorporating Retrieval-Augmented Generation (RAG) technology, the system can mount the merchant's "standard script library" and "return/exchange rules" as an external knowledge base. It automatically generates accurate, compliant, and highly anthropomorphic replies that mimic the merchant's exclusive tone, significantly reducing the communication costs of manual customer service.
Deep Review Sentiment Analysis: Combining asynchronous concurrency mechanisms with the text comprehension capabilities of LLMs, the system can efficiently perform data cleaning, multi-dimensional sentiment classification (positive/neutral/negative), and automatically extract high-frequency product defect entities (e.g., "damaged packaging," "easily broken clasps") from massive raw reviews.
Ultimately, all processed data results and business insights are clearly presented through an intuitive data visualization dashboard. This project not only delivers a highly available engineering prototype system but also provides data-driven product iteration recommendations for merchants, demonstrating significant academic research and practical application value.
